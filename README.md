# telegram-desktop-solarized
A solarized-dark theme for telegram-desktop.

## Installation
- Copy it to your desired location.
- In telegram-desktop got to
	- settings
	- choose from file
	- select solarized-dark.tdesktop-palette
	- open
	- that's it, you selected the theme!
